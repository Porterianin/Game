# Приключения в большом городе

Минимальный каркас визуальной новеллы на Ren'Py о девушке Насте, которая приехала в большой город поступать в медицинский институт. В проекте уже настроены базовые экраны, карта города и домашняя локация.

## Структура
- `game/options.rpy` — название игры и базовые настройки Ren'Py.
- `game/script.rpy` — основной сценарий, персонажи, переменные, переходы по локациям.
- `game/screens.rpy` — HUD со статами, карта города, окно достижений.

## Быстрый старт
1. Откройте проект в Ren'Py и запустите игру.
2. После короткого вступления вы окажетесь в съемной квартире. Используйте кнопку «Карта города» для перехода между локациями (университет, аптека, парк) и следите за параметрами Насти.
