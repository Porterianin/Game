# Приключения в большом городе

Минимальный каркас визуальной новеллы на Ren'Py о девушке Насте, которая приехала в большой город поступать в медицинский институт. В проекте уже настроены базовые экраны, карта города и домашняя локация, а также пролог «Дорога в большой город» с первыми сценами пути Насти.

## Структура
- `game/options.rpy` — название игры и базовые настройки Ren'Py.
- `game/script.rpy` — основной сценарий, персонажи, переменные, переходы по локациям.
- `game/screens.rpy` — HUD со статами и состояниями, карта города, окно достижений и профиль персонажа.

## Быстрый старт
1. Откройте проект в Ren'Py и запустите игру.
2. Просмотрите пролог с пятью сценами (сборы в деревне, наряд, биография, эпизод с ветром у автобуса и поездка). После вступления вы окажетесь в съемной квартире. Используйте кнопку «Карта города» для перехода между локациями (университет, аптека, парк), следите за параметрами Насти (энергия, сытость, чистота, возбуждение, деньги, репутация) и открывайте «Статус», чтобы изучить характеристики, одежду и инвентарь.

## Проверка запуска локально
1. Скачайте и распакуйте Ren'Py (например, в `~/renpy`).
2. Укажите путь к каталогу с `renpy.sh` через переменную `RENPY_HOME` и запустите проверку:
   ```bash
   RENPY_HOME=~/renpy ./scripts/run-local.sh lint
   ```
   Если нужно сразу открыть игру, используйте:
   ```bash
   RENPY_HOME=~/renpy ./scripts/run-local.sh launch
   ```
   Для последовательной проверки и запуска:
   ```bash
   RENPY_HOME=~/renpy ./scripts/run-local.sh both
   ```
